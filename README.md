# 🚀 Mon Projet Fullstack

Une application web structurée en deux parties : un backend performant avec **Node.js** et un frontend léger en **HTML/CSS/JS**.

---

## 📂 Structure du Répertoire

* **`/backend`** : Serveur API (Express.js) et connexion base de données (MongoDB).
* **`/frontend`** : Interface utilisateur (fichiers statiques).

---

## 🛠️ Technologies Utilisées

* **Backend** : Node.js, Express, Mongoose (MongoDB).
* **Frontend** : HTML5, CSS3, JavaScript (Vanilla).
* **Versionnage** : Git & GitHub.
